# SandiaPop ReactNativeAPP

    Demo App for posting item to sell,
    Upload your images to the server
    and get listed in the home screen.

## Using:

### Client Side

- Expo
- React Native
- Typescript
- UseReducer

### Server Side

- NodeJs
- Express
- Moongose
- S3-AmazonWebService
- API Gateway-AmazonWebService
